# Projet AlgoBox TP3

Ce projet contient tous les exercices du TP3 réalisés avec AlgoBox.

## Comment l’utiliser
1. Cloner le projet :
   git clone https://github.com/ton-utilisateur/projet-algobox.git
2. Ouvrir les fichiers .alg avec AlgoBox
3. Faire les modifications nécessaires

## Collaboration
- Avant de commencer : git pull origin main
- Après modification : git add . → git commit -m "message" → git push origin main
